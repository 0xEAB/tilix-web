Contribute

Terminix is localized using Weblate, please visit the Weblate hosted Terminix transations site in order to assist with translations, please do not submit direct pull requests to the repository for translations.


If you are having issues with Terminix, feel free to open issues here in Github as necessary. Developers and users can also be found on IRC on the freenode network in the #terminix room.
