# Terminix website
Website for the Terminix project

#### Build:
1- Clone the repository 
```bash
git clone https://github.com/gnunn1/terminix-web/
```
2- Go the cloned directory and run Jekyll server
```bash
cd ./terminix-web && jekyll serve
```
